String formatting mini-language

  • How to set number of decimals: {:.2f}
  • How to make a percent: {:.1%}
  • Right-justify numbers in a certain sized space: {:>10}
  • Right-justify costs: ${:>10,.2f}

Running a simple server from the command line: python -m http.server 8000. To access the served files, open a browser and go to localhost:8000. (You can choose whatever port you'd like by changing the number from 8000 to something else.)
